# 555 Timer LED Blinker PCB using KiCad 9

## Overview

This project implements a 555 Timer based LED Blinker circuit and its PCB layout using **KiCad 9.0**. The circuit operates in astable mode, continuously switching the LED ON and OFF at a fixed interval.

This project was designed to learn the complete PCB design workflow, including schematic capture, PCB layout, routing, design rule checking (DRC), and 3D visualization.

---

## Features

- Designed using KiCad 9.0
- 555 Timer in Astable Mode
- Single Layer PCB
- Through-Hole Components
- LED Blinking Circuit
- DRC Error Free
- 3D PCB Visualization
